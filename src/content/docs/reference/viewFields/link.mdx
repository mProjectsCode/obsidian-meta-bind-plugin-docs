### Description

A _link_ view field only accepts a single bind target as it's content and will automatically link to the targets value.
The following property types are supported:

- a wiki-link to a note (e.g. `[[note]]`)
- a markdown link (e.g. `[link](https://example.com)`)
- a string will get turned into a wiki-link (e.g. `note` -> `[[note]]`)
- a URL will get turned into a markdown link (e.g. `https://example.com` -> `[https://example.com](https://example.com)`)
- a list of the above (e.g. `[[note]], [link](https://example.com), note2`)
