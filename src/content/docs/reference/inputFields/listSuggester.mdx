### Description

A _list suggester_ input field allows you to add and remove elements to a list, using a <InputFieldLink type={InputFieldType.SUGGESTER}></InputFieldLink> input.

Elements can be moved and deleted from the list by right-clicking on the element (or long-pressing on mobile) and selecting the appropriate action.
