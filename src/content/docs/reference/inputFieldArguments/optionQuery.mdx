The `optionQuery` argument lets you add multiple options to suggesters based on a [dataview data source](https://blacksmithgu.github.io/obsidian-dataview/reference/sources/).

Note that the <InputFieldLink type={InputFieldType.IMAGE_SUGGESTER}></InputFieldLink> only supports providing a `"path/to/folder"`, whereas other suggester variations fully support any dataview data source.

### Values

<InputFieldArgumentAllowedInputFields type={InputFieldArgumentType.OPTION_QUERY}></InputFieldArgumentAllowedInputFields>

### Examples

```meta-bind
INPUT[suggester(optionQuery(#tag)):bind_target
```

```meta-bind
INPUT[suggester(optionQuery("path/to/folder")):bind_target]
```

```meta-bind
INPUT[imageSuggester(optionQuery("path/to/folder")):bind_target]
```
