The `addLabels` argument lets you add a labels to a <InputFieldLink type={InputFieldType.SLIDER}></InputFieldLink> or <InputFieldLink type={InputFieldType.PROGRESS_BAR}></InputFieldLink>.
The value passed to this argument must either be `true` or `false`. If no value is provided, `true` will be used for the value.

<InputFieldArgumentAllowedInputFields type={InputFieldArgumentType.ADD_LABELS}></InputFieldArgumentAllowedInputFields>

### Examples

```meta-bind
INPUT[slider(addLabels):bind_target]
```

```meta-bind
INPUT[slider(addLabels(true)):bind_target]
```
