The `stepSize` argument lets specify a step size value for a <InputFieldLink type={InputFieldType.SLIDER}></InputFieldLink> or <InputFieldLink type={InputFieldType.PROGRESS_BAR}></InputFieldLink> input.
The value passed to this argument must be a number.

### Values

<InputFieldArgumentValuesInfo type={InputFieldArgumentType.STEP_SIZE}></InputFieldArgumentValuesInfo>

<InputFieldArgumentAllowedInputFields type={InputFieldArgumentType.STEP_SIZE}></InputFieldArgumentAllowedInputFields>

### Examples

```meta-bind
INPUT[slider(stepSize(2)):bind_target]
```

```meta-bind
INPUT[slider(stepSize(0.1)):bind_target]
```
