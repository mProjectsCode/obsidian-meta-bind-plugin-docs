The `maxValue` argument lets specify a maximum value for a <InputFieldLink type={InputFieldType.SLIDER}></InputFieldLink> or <InputFieldLink type={InputFieldType.PROGRESS_BAR}></InputFieldLink> input.
The value passed to this argument must be a number.

### Values

<InputFieldArgumentValuesInfo type={InputFieldArgumentType.MAX_VALUE}></InputFieldArgumentValuesInfo>

### Allowed Input Fields

<InputFieldArgumentAllowedInputFields type={InputFieldArgumentType.MAX_VALUE}></InputFieldArgumentAllowedInputFields>

### Examples

```meta-bind
INPUT[slider(maxValue(10)):bind_target]
```

```meta-bind
INPUT[slider(minValue(-10), maxValue(10)):bind_target]
```
